About Heart AI

Project By 1) Mariya Patel 2) Deepika Jangam 3) Inzila Shaikh
Purpose: Heart AI offers simple, accurate heart disease prediction and BMI calculation for everyone.
We use modern AI/ML, and a secure Supabase backend for user management and profile data.
Technology:
  • Vanilla JavaScript (no frameworks)
  • Mobile-first, responsive dark UI
  • PWA install - offline first, auto updates, version checks
  • Supabase (Postgres) backend for user/db operations
  • BMI & prediction history stored locally and/or server
